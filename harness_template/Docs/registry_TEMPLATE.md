# {{PROJECT}} 수치 정본 레지스트리 (registry)

> **이 파일이 수치의 단일 정본이다.** 다른 모든 문서(EXPERIMENTS·발표·논문·세션 메모리)의 수치는 이 표의 **★확정 행에서만 인용**한다.
> 개명·이동: `{{문서 루트}}/experiments/{{registry}}.md` 로 배치 (wiki-structure.md 트리 참조).

## 등급제 (행마다 필수)

| 등급 | 의미 | 인용 가능 여부 |
|---|---|---|
| ★확정 | 사용자 검토를 통과한 확정 수치 | **인용 가능 (유일)** |
| ○후보 | 측정 완료, 사용자 검토 대기 | 인용 금지 — 검토 후 승격 |
| △이슈 | 측정에 결함/의문 존재 (재측정·폐기 대상) | 인용 금지 |

## 등록 규칙

- **등록·승격(→★확정)은 사용자 검토 후에만** — 에이전트 단독 등록 금지 (experiments/RULES.md)
- 행은 수정하지 않고 **새 행 추가 + 옛 행 등급 강등**으로 이력 보존
- 원본 데이터 경로(로그·bag·캡처)를 비고에 반드시 남긴다 — 재현 불가 수치는 등록 불가
