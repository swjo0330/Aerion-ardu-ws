# ⑥ 세션 재시작 회상 (기억→미결 원장→대시보드)

**언제 쓰나**: 새 세션 시작 시, 이전 상태를 복원하고 오늘 할 일을 정할 때.

## 프롬프트

```
세션 재개. 다음 순서로 상태를 복원한 뒤 오늘 작업을 제안하라:
1. 세션 기억 — {{세션 메모리 위치}}의 최신 항목
   <!-- 예: project_session_* 최신 파일, 재개 체크리스트 절 -->
2. 미결 원장 — {{구현 기록/미결 리스트 파일}}
   <!-- 예: implementation-notes.md의 [빈칸][가정][결정][이탈][테스트] -->
3. 대시보드 — {{프로젝트 총괄 현황판}}
   <!-- 예: EXPERIMENTS.md 진행판·실행일 게이트 -->
그 다음: 환경 변동 확인({{환경 체크 항목}} <!-- 예: IP/네트워크 -->) →
미커밋 변경 확인 → "지난 세션 완료 / 이월 / 오늘 큐" 3분류로 보고.
세션 유형이 여럿이면({{세션 체계}}) 자기 유형의 진입점만 읽고 타 영역 수정 금지.
```

**실증**: 프로젝트 CLAUDE.md 세션 체계 — 총괄 세션 진입점 "세션 메모리→미결 원장→현황판", 문서 세션은 별도 handoff 진입점, 재부팅 후 재개 체크리스트로 실사용.
