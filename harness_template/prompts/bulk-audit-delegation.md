# ⑦ 문서 대량 감사 위임

**언제 쓰나**: 인용·수치·로그·인터페이스 등 항목이 수십 건 이상인 문서/데이터를 전수 검증할 때.

## 프롬프트

```
{{감사 대상}} 전수 감사를 수행하라 (총 {{N}}건).
- 분할: 도메인/구간별로 에이전트 병렬 배분 → 합성 → 적대검증 → 수정 순서
- 판정 기준: 각 항목에 근거 위치(파일:줄 / 원 소스) 명시 —
  근거를 못 대는 항목은 "미확인" 부록으로 분리(삭제하지 않음)
- 대조 축 {{K}}개를 교차 검증 <!-- 예: 코드↔로그↔문서↔실측 4단 대조 -->
- 산출: 확정/정정/미확인 3분류 리포트 + 정정은 심각도순 + 원본 반영은
  불가역 문서면 사용자 검토 후에만
```

**실증**: 서지 54건 전수검증(허구 인용 4건 정정·출판본 승격 7건), 로깅 상태 감사 4단 대조(critical 3건 수정+잠복 버그 회복), 인터페이스 명세 80건 전 항목 파일:줄 근거+미확인 33건 부록 분리.
